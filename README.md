# C# Console Learning Journey

This repo is created to better understand the fundamentals for Unity.

My goal is to be able to explain the reasons to myself whenever I ask "Why?" while coding. To avoid future technical debt when I eventually dive into Unity's core (which is written in C++), I am trying to build a solid foundation by practicing with pure C# on the Console.

I prefer understanding the logic rather than doing everything by copy-pasting or memorizing.

Each file here is written to reinforce a specific topic.

---

### Türkçe

Bu repo, Unity'nin temellerini daha iyi anlamak için oluşturulmuştur.

Buradaki amacım; kod yazarken "Neden?" sorusunu sorduğumda, sebebini kendi kendime açıklayabilmektir. İleride Unity'nin derinliklerine (C++ ile yazılmış çekirdeğine) indiğimde teknik borçlu olmamak için; Console üzerinde saf C# ile pratikler yaparak temelleri sağlam atmaya çalışıyorum.

Her şeyi kopyala-yapıştır (copy-paste) veya ezbere yapmaktansa, mantığını anlayarak yapmayı tercih ettim.

Buradaki her dosya (file), belli bir konuyu pekiştirmek için yazılmıştır.
